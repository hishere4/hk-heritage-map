# HeritageHK 數據豐富化指南

## 📝 建議新增內容欄位

### 現有欄位
- [x] name（名稱）
- [x] description（簡介）
- [x] history（歷史）
- [x] cultural_significance（文化意義）
- [x] fun_facts（趣聞）
- [x] recognition_year（列入年份）
- [x] address（地址）

### 建議新增欄位

#### 1. `highlights` (Array of strings) - 必睇亮點
```json
[
  "欣賞傳統粵劇面譜化妝過程",
  "聆聽獨特嘅『梆黃』唱腔",
  "觀賞水袖功同做手身段",
  "認識十大行當角色分類"
]
```

#### 2. `how_to_experience` (String) - 體驗方式
```
西九戲曲中心定期舉辦粵劇演出，可於城市售票網購票。
建議首次體驗者選擇經典劇目如《帝女花》香夭。
亦可參加戲曲導賞團，深入了解粵劇文化。
```

#### 3. `best_time` (String) - 最佳時間
```
全年均有演出，但農曆新年期間會有特別賀歲劇目。
每年 11 月戲曲節更有多場國際級演出。
建議避開暑假兒童劇場時段。
```

#### 4. `tips` (Array of strings) - 體驗貼士
```json
[
  "演出約 3 小時，建議預留充足時間",
  "場內禁止攝影錄音",
  "可提早 30 分鐘到場閱讀劇目介紹",
  "穿著端莊，避免短褲拖鞋"
]
```

#### 5. `transportation` (String) - 交通指南
```
港鐵柯士甸站 F 出口，步行約 5 分鐘。
或乘坐巴士至「戲曲中心」站。
附近設有停車場，但演出日經常滿位。
```

#### 6. `nearby_attractions` (Array) - 附近景點
```json
[
  { "name": "西九文化區", "distance": "步行 3 分鐘" },
  { "name": "香港故宮", "distance": "步行 8 分鐘" },
  { "name": "M+ 博物館", "distance": "步行 10 分鐘" }
]
```

#### 7. `master_quotes` (Array) - 傳承人語錄
```json
[
  {
    "quote": "粵劇係我哋嘅根，要傳承落去。",
    "master": "阮兆輝",
    "title": "資深粵劇演員"
  }
]
```

#### 8. `practical_info` (Object) - 實用資訊
```json
{
  "opening_hours": "視乎演出時間",
  "ticket_price": "$100-$480",
  "booking_url": "https://urbtix.hk",
  "contact_phone": "+852 2200 0020",
  "accessibility": "設有輪椅座位"
}
```

## 🎯 內容優先級

### 高優先（立即影響用戶體驗）
1. Highlights（必睇亮點）- 快速吸引興趣
2. How to experience（體驗方式）- 轉化為預訂
3. Best time（最佳時間）- 幫助規劃行程
4. Tips（體驗貼士）- 提升體驗質素

### 中優先（豐富內容）
5. Transportation（交通指南）
6. Nearby attractions（附近景點）
7. Master quotes（傳承人語錄）

### 低優先（錦上添花）
8. Practical info（實用資訊）

## 📝 內容寫作指南

### 語調
- 親切、專業、有權威性
- 使用香港本地用語
- 避免太學術化

### 長度
- Highlights：每點 10-20 字
- How to experience：50-100 字
- Tips：每點 8-15 字
- History / Cultural significance：100-200 字

### 格式
- 使用表情符號增加趣味性
- 善用 bullet points
- 重要資訊用粗體
